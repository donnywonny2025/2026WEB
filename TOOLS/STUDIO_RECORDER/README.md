# Studio One — Professional Master Capture

A high-fidelity system audio recording tool built for macOS. It captures exactly what is playing on your computer (system output) without using the microphone.

## Features
- **Real WAV Recording**: Captures RAW 44.1kHz stereo audio and encodes it into standard RIFF/WAVE format. Works perfectly with **macOS QuickLook (Spacebar)**.
- **Multicolored Visualizer**: A beautiful dynamic spectrum that reacts to the audio signal.
- **Professional Headroom**: Meter levels are tuned to stay in the "Safe Zone" to prevent digital clipping/hot levels.
- **Direct Save**: Immediate file download upon stopping the recording.

## Location
The tool is located at:
`/Volumes/WORK 2TB/WORK 2026/STUDIO_RECORDER/studio-recorder.html`

## How to Use
1. Open `studio-recorder.html` in a modern browser (Chrome or Edge recommended).
2. Click **Link System Audio**.
3. In the browser popup, select the source (Window or Tab).
4. **CRITICAL**: Check the **"Share Audio"** box at the bottom of the dialog.
5. Click **Record** to start capture.
6. Click **Stop** to immediately save the high-fidelity `.wav` file.
